# 설명
**웹 뷰 위젯**을 이용해서 만든 **웹 앱**입니다.

# 주요 기능
1. 실행하면 **상단의 'Github_Site' 라는 문구의 앱 바**와 함께 밑에 **깃허브 사이트**가 실행된다.
2. 이때, **상단 앱바 오른쪽에 있는 집 모양 버튼**을 누르면 깃허브 사이트가 **새로고침**된다.
